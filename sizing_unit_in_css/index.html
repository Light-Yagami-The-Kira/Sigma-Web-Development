Sizing Units in CSS

There are different sizing units in CSS. Some of them are:

  • px ---> 1/96 of an inch
  • em ---> relative to the font-size of the element
  • rem ---> relative to the font-size of the root element
  • % ---> relative to the font-size of the root element
  • vh ---> relative to the 1% of the viewport's height
  • vw ---> relative to the 1% of the viewport's width
  • vmin ---> relative to the 1% of the viewport's smaller dimension
  • vmax ---> relative to the 1% of the viewport's larger dimension
